# DBJARVIS

국내주식 자동매매 시스템

## 기능

- 4단계 자동매매 전략 (단타 / 스윙 / 포지션 / 장투)
- NXT 거래소 연동 + 시간대별 자동 라우팅
- 테마/종목 마스터 자동 갱신
- 텔레그램 봇 연동
- 웹 대시보드
- 리스크 관리 (손절/익절/트레일링/자금격리)

## 설치

1. [Releases](https://github.com/systrader/DBJARVIS/releases) 에서 최신 ZIP 다운로드
2. 압축 해제
3. `settings_launcher.exe` 실행하여 설정
4. `run_all.exe` 실행

## 자동 업데이트

`updater.exe`가 하루 4회 자동으로 새 버전을 확인합니다.
수동 확인: `updater.exe --force`
